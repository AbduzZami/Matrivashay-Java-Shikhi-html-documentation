IDE তে জাভা শুরু করা

আমি IDE হিসেবে JetBrains এর IntelliJ ব্যাবহার করেছি এই বই এর কোড লিখার সময়। তো আমি এটা দিয়েই দেখাচ্ছি প্রোজেক্ট খোলা। যে কেও যেকোনো IDE ব্যাবহার করতে পারে। 


শুরুতেই IntelliJ Idea ওপেন করি।





এর পর একটি Pannel আসবে। সেখানে New Project এ ক্লিক করি।

 


আবার ও একটি প্যানেল আসবে। সেখানেও Next এ ক্লিক করি।



এর পরেও আরেকটি প্যানেল আসবে সেখানেও Next বাটন এ ক্লিক করি।



সব শেষে একটি উইন্ডো আসবে যেখানে ফাইলের একটি নাম এবং ফাইল লোকেশন দিয়ে Finish বাটনে ক্লিক করলেই নতুন প্রোজেক্ট তৈরি হয়ে যাবে।


এরপর নিচের মত একটি উইন্ডো আসবে। 


এখন বামে উপরে প্রোজেক্ট নেইম এর নামে যেই ফোল্ডারটি দেখা যায় তাকে expand করি। Expand করার জন্য > এই চিহ্নে ক্লিক করতে হবে। 


এরপর src নামের ফোল্ডার এ mouse এর right বাটনে ক্লিক করলে যে পপ আপ আসবে তাতে  new এ কার্সর রাখলে আরও একটি পপ আপ আসবে। 



সেখানে java class এ ক্লিক করলে আরেকটি পপ আপ আসবে। 

ওই পপ আপ এর class অপশন এ সিলেক্টেড থাকা অবস্থায়  java class এর একটি নাম দিয়ে Enter দিলেই একটি নতুন জাভা ক্লাস খুলে যাবে।


এটি হচ্ছে একটি জাভা ক্লাস। এখানেই আমরা যাবতীয় কোড লিখবো। 


জাভা ক্লাসের ভিতরে একটি মেইন মেথড নিতে হবে। public static void main(String[] args){} এটি ই হচ্ছে মেইন মেথড। এটি লেখার জন্য IntelliJ Idea এ একটি শর্ট কাট আছে। এটি হল psvm লিখে tab / enter বাটনে প্রেস করলেই মেইন মেথড তৈরি হয়ে যাবে। 



মেইন মেথডের ভিতরে বাইরে সব খানেই কোড লিখতে হবে। আমরা বেসিক কোডগুলো মেইন মেথডেই লিখবো। 






কোড লেখার পর আউটপুট দেখার জন্য ওই ফাইলেই কোড লেখার জন্য যে জায়গা আছে সেখানেই মাউসের রাইট বাটনে ক্লিক করলে একটি পপ আপ আসবে। সেখানে Run File Name নামে একটি অপশন আছে। সেখানে ক্লিক করলে জাভা ফাইলটি রান হবে।

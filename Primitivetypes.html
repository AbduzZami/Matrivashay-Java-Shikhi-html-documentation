  1. ডাটা টাইপস

Primitive types:

Primitive type variable ডিক্লেয়ার করার সাধারণ নিয়ম:

Type variable = value;


byte:

byte s = 127;


এখানে s হচ্ছে একটি byte টাইপ variable যার ভ্যালু হচ্ছে 127।

byte 1 byte জায়গা দখল করে।

-128 থেকে 127 পর্যন্ত সংখ্যা রাখা যায়।


short:

short x = 10000;


এভাবে short ডিক্লেয়ার করতে হয়। short 2 byte জায়গা দখল করে। 

-32,768 থেকে 32,767 পর্যন্ত সংখ্যা রাখা যায়।


int:

int z = 1000000000;


এভাবে int ডিক্লেয়ার করতে হয়। int 4 byte জায়গা দখল করে। 

-2,147,483,648 থেকে 2,147,483,647 পর্যন্ত সংখ্যা রাখা যায়।


long:

long y = 1000000000000000000L;


এভাবে long টাইপ ভেরিয়েবল ডিক্লেয়ার করতে হয়। সংখ্যাটির শেষে capital L দিতে হবে অন্যথায় জাভা কম্পাইলার এটিকে বুঝতে পারবেনা। কম্পাইলার ধরে নিবে এটি একটি int.

long 8 byte জায়গা দখল করে। 

-9,223,372,036,854,775,808 থেকে 9,223,372,036,854,775,807 পর্যন্ত সংখ্যা রাখা যায়।


double:

double x = 5.34;


ভগ্নাংশ গুলো কে double এ রাখতে হয়। double 8 byte জায়গা দখল করে।

15 decimal digits পর্যন্ত সংখ্যা রাখা যায়।


float:

float x = 5.34f;


ভগ্নাংশ গুলো কে float এও রাখা যায়। float 4 byte জায়গা দখল করে। float এর বেলায় সংখ্যাটির শেষে small f অথবা capital F দিতে হবে অন্যথায় জাভা কম্পাইলার এটিকে বুঝতে পারবেনা। কম্পাইলার ধরে নিবে এটি একটি double. কারন ভগ্নাংশ পেলেই জাভা কম্পাইলার ধরে নেয় এটি একটি double।

6 থেকে 7 decimal digits পর্যন্ত সংখ্যা রাখা যায়।


boolean:

boolean b = false;


boolean এ শুধুমাত্র true/false রাখা যায়। এটি মাত্র 1 bit জায়গা দখল করে।


char:

char c = ‘A’;


char এ একটি character রাখা যায়। character কে single quotation (‘ ‘) এর ভিতরে রাখতে হয়। এটি 2 byte জায়গা দখল করে।
